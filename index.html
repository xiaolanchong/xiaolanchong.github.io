Японский язык

  • Кандзи с определениями из словаря Яркси.

  • Цепи дзёё кандзи. Цепь - серия кандзи с общим фонетиком (например, 列-例-烈-裂).
  • Библиотека японской лит-ры.

Китайский язык

  • Списки ханьцзы с определениями из словарей БКРС и CEDICT.

Корейский язык

  • Корейские слова из ханчи.
  • Кор-анг. словарь среднего уровня издательства Tuttle.
  • Библиотека корейской лит-ры.

Разное

  • Система повторения слов ин. языка по системе Нильса Иверсена (слов - значение - слово).
